"&nbsp;" coloca um espaço entre textos.
"<wbr/>" cria uma quebra de linha em uma palavra composta (ou endereço de e-mail por exemplo) qdo necessário.
"&shy;" quebra uma palavra utilizando hifenização.
"<em></em>" é o mesmo que "<i></i>" coloca o texto em itálico.
"<del></del>" é o mesmo que "<s></s>" coloca uma linha sobre o texto informando que ele foi apagado.
"<code></code>" mostra trechos de código embutido e deve ser combinado com "<pre></pre>".
"<sup></sup>" para texto sobreescrito e "<sub></sub>" para texto subescrito
